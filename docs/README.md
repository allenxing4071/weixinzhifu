## 📝 文档维护规范

### 命名规范
- **全部使用中文命名**
- **目录以数字编号 + 中文描述**
- **文件以中文描述 + .md后缀**

### 更新规范
- **新增文档**：放入对应功能目录
- **版本管理**：重要变更需要版本标记
- **状态跟踪**：标明文档状态（草案/待评审/已确认）

### 文档状态说明
- ✅ **已完成** - 内容完整，与实际功能匹配
- 🔄 **更新中** - 正在根据实际情况调整
- ⚠️ **待确认** - 需要进一步验证或评审
- 📋 **参考** - 仅作参考，可能不是最新状态
